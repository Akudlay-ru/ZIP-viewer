# ZIP Content Viewer

ZIP Content Viewer — локальный плагин для Obsidian, который добавляет просмотр содержимого ZIP-архивов прямо внутри Obsidian.

Плагин нужен для случаев, когда архив лежит в хранилище Obsidian, но его не хочется каждый раз открывать внешней программой, распаковывать вручную и терять связь с исходным файлом.

## Возможности

- Открытие `.zip` из дерева файлов Obsidian.
- Просмотр структуры файлов и папок внутри архива.
- Поиск / фильтрация файлов по имени.
- Отображение размера, метода сжатия и даты изменения.
- Предпросмотр текстовых файлов.
- Предпросмотр Markdown, JSON, CSV, HTML, CSS, JS, Python, YAML, XML и LOG.
- Форматирование JSON для удобного чтения.
- Предпросмотр изображений PNG, JPG, GIF, WEBP и SVG.
- Копирование текста из предпросмотра.
- Извлечение выбранного файла.
- Извлечение и открытие файла двойным кликом.
- Открытие ZIP во внешней программе Windows через кнопку `Open ZIP`.

## Куда извлекаются файлы

Файлы извлекаются в папку:

```text
_zip_extract/<имя архива>/<путь файла внутри архива>
```

Так сохраняется исходная структура архива, а извлечённые файлы не превращаются в свалку с одинаковыми именами.

## Ограничения

- Поддерживаются только ZIP-архивы.
- Поддерживаются методы сжатия Stored и Deflate.
- ZIP64 не поддерживается.
- Зашифрованные архивы не поддерживаются.
- RAR и 7z не поддерживаются.
- PDF внутри ZIP можно извлечь и открыть через установленный PDF-просмотрщик.

## Установка вручную

1. Закрыть Obsidian.
2. Скопировать папку `zip-content-viewer` в папку `.obsidian/plugins/`.
3. Проверить структуру:

```text
.obsidian/plugins/zip-content-viewer/manifest.json
.obsidian/plugins/zip-content-viewer/main.js
.obsidian/plugins/zip-content-viewer/styles.css
```

4. Открыть Obsidian.
5. Перейти в `Settings → Community plugins`.
6. Включить `ZIP Content Viewer`.
